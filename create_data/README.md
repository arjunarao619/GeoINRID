# Dataset and Embedding Extraction

We run several experiments that require the use of external Earth embeddings available through APIs. We include scripts to query AlphaEarth embeddings and regional Sentinel-2 image datasets. 

To view AlphaEarth embedding extraction details, see `create_data/alphaearth_retrieval/README.md`

To view Sentinel-2 regional dataset generation instructions, see `create_data/s2_retrieval/README.md`
